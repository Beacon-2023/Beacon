# Beacon
지역별 실시간 재난 속보 및 안전 대응 API 서버 입니다.

## 프로젝트 목표
*  지역별로 실시간 가공된 재난 속보를 제공 하는 것이 목표입니다.
*  재난 발생시 대피소, 행동 요령 등 안내를 푸시 알람 및 지도 앱 서비스를 통해 전달 하는 것이 목표입니다.
